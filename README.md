# Claude Prompt Generator

A web application for generating structured prompts optimized for Claude, with best practices for XML tags, system prompts, and prompt engineering patterns.

Also includes a landing page concept for [TalentSpottingAI](https://www.talentspottingai.com) with pages for Students, Employers, and Universities.

## Stack

- **Framework:** Vite + React 18
- **Language:** TypeScript
- **UI:** shadcn/ui + Tailwind CSS
- **Prompt Generator:** React (CRA) in `Projects/Prompt Generator/`

## Structure

```
/                          # TalentSpottingAI landing page (Vite + React)
/Projects/Prompt Generator # Claude prompt generator tool (CRA)
/landing-page              # Static HTML landing page prototype
```

## Built with

Built using Claude Code as an early exploration of AI-assisted development and Claude API prompt patterns.
