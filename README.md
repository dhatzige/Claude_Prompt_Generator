# Claude Prompt Generator

An early exploration project from my first months of AI-assisted development. Contains a prompt generator tool for structuring Claude API prompts with XML tags and best practices, plus a landing page prototype for [TalentSpottingAI](https://www.talentspottingai.com).

**Context:** This was one of my first coding projects, built when I was learning to vibecode. It is rough and not representative of my current work. For a better example, see [CV-EXTRACTOR](https://github.com/dhatzige/CV-EXTRACTOR) or visit [TalentSpottingAI](https://www.talentspottingai.com) (223K LOC, built with Claude Code).

## Structure

```
/                          # TalentSpottingAI landing page prototype (Vite + React)
/Projects/Prompt Generator # Claude prompt generator tool (CRA + React)
/landing-page              # Static HTML landing page prototype
```

## Stack

- **Framework:** Vite + React 18 / CRA
- **Language:** TypeScript
- **UI:** shadcn/ui + Tailwind CSS

## Built with

Built using AI-assisted development as a learning exercise while exploring Claude API prompt engineering patterns.
